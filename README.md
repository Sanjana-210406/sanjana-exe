# Sanjana.exe

An interactive cinematic portfolio experience guided by an AI companion named Elyra.

This project is not being built as a traditional portfolio website.  
Instead of simply showcasing projects and skills through static sections, the goal is to create a living digital world where visitors explore memories, ideas, creations, and ambitions through immersive storytelling and interaction.

The experience begins with Elyra — a futuristic guide entity that introduces visitors to my universe, reacts to interactions, and slowly unfolds different chapters of my journey as a developer.

## Vision

I wanted to build something that feels memorable.

Not just:
- another scrolling portfolio
- another template clone
- another “About Me” page

But an experience that feels alive.

Sanjana.exe combines:
- storytelling
- frontend development
- interaction design
- animations
- immersive UI/UX
- AI-inspired experiences

into one evolving project.

## Current Status

This project is currently being built from scratch while I learn and improve my frontend development skills step by step.

The idea is to grow the project alongside my learning journey.

### Planned Features

- AI companion guide (Elyra)
- Interactive storytelling sections
- Cinematic UI animations
- Memory-based project showcase
- Dynamic chatbot interactions
- Immersive futuristic atmosphere
- Responsive experience across devices

This project is being built with a strong focus on:
